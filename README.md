## 서비스 추가 개선사항
### 레디스 장애시 커플 메시지 동기화 : QueueService.java
1. ZSET 자료구로를 사용해서 커플 메시지 큐를 통해 데이터 동기화
2. 만약 장애 발생으로 레디스 접근을 못할 시 로컬 캐시를 통해 데이터 동기화

### 레디스 장애시 데이터베이스 동기화
1. 커플 데이터 동기화를 위해 레디스 SET HASH 구조 사용하고 이를 주기적으로 데이터베이스에 동기화
2. 만약 장애 발생으로 레디스 접근을 못할 시 업데이트가 발생하면 바로 데이터베이스에 업데이트 사항 반영
